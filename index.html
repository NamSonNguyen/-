Nam is an assistant professor at UW-Stout. Nam's research interests are cloud computing, big data analytics, mobile computing, focusing on building scalable, improving the high-performance system.

Currently, Nam is pursuing is exploring security issues with automatic speech recognition or ASR.

Projects

Speech Enhancement with Noise Filtering

Eliminating environmental sound (noise filter) and extracting only the speaker's voice

End to End Deep Neural Network ASR

A deep neural network that functions end-to-end automatic speech recognition (ASR) pipeline

Launching Big Data Platforms in IoT

A cross-layer framework bewteen application and link layer
